# Team24_Fourth Project

천재교육 풀스택 Java 과정 2기 4차 프로젝트입니다.  
<주제: 해법강의 시범강의 웹 서비스 구현>  

<strong>SmartHabeop</strong>은 학생과 학부모가 시범강의를 통해 결제 후 수강 할 수 있는 웹서비스 입니다.
